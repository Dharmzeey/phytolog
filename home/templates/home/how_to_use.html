How to use

General Search

This search returns any and all specie that matches the keyword being searched.

Local Name

This search checks for the local name being searched only and returns the specie and specie loacation for which contains the local name being searched. The local name is the name indigenous name amongst the yoruba people, but some might accomodate other ethnic languages

Scientific Name

This search checks for the scietific name being searched only and returns the specie and specie loacation for which contains the Scientific name being searched

Coordinate

This requires and works well with gps enabled sevices, mostly mobile devices. You will need to accept location request then the Coordinate of your location will be displayed in the input column. This will display only the trees arround your location. This can be helpful if you want to check a tree name where you are currently.

Common Name

This search checks for the common name being searched only and returns the specie and specie loacation for which contains the common name being searched

Location Name

This search checks for the Location name being searched only and returns the specie and specie loacation for which contains the Location name being searched. It is adviseable to search for a single keyword and a popular place around where you are trying to get the tree name.

Special Places

I came up with the idea of this special Places because not everywhere do we have trees standing on their own or appearing in ones or twos. We have places with clusters of trees which can not be accessed individually. So the places are taken note and described based on the consititents of the trees.
